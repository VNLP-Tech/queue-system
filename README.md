# queue-system
Base queue for call system

### Installation
```bash
# install dependencies
npm i
```

### Start
```bash
# run server to serve queue
npm start
```

### Add job
```bash
# can adjust data job in `test/add-job.js`
npm run job
```

### Setting maxCCU
You can change maxCCU in DB to increase/decrease CCU of system.
